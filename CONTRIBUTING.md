# Contributing

Thank you for contributing!

- Fork the repo and create feature branches (feature/... or fix/...).
- Install dev dependencies:
  - python -m venv .venv
  - source .venv/bin/activate
  - pip install -r requirements-dev.txt
  - pip install -e .
- Run linters and formatters:
  - pre-commit install
  - pre-commit run --all-files
- Run tests: `pytest`
- Make small, focused PRs with clear descriptions and testing instructions.
- Follow the code of conduct and be respectful.
